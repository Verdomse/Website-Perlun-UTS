Sebutkan langkah-langkah secara umum untuk membuat model struktur Frame dengan SAP2000 dan hubungannya dengan matriks kekakuan, beban dan displacement struktur?
Langkah-Langkah Pemodelan Frame di SAP2000
  • Pengaturan Satuan & New Model: Buka SAP2000 → klik File > New Model → pilih satuan sistem (kN, m, °C) di pojok kiri bawah.
  • Pilih Template: Pilih template kosong (Blank) atau template frame yang sesuai (2D Frame, 3D Frame). Tentukan jumlah span dan tinggi lantai.
  • Definisi Material: Define > Materials → Add New Material → input properti: modulus elastisitas (E), berat jenis (γ), kuat leleh (fy untuk baja) atau kuat tekan (f'c untuk beton).
  • Definisi Penampang (Section): Define > Section Properties > Frame Sections → Add New Section → pilih tipe (I/Wide Flange, Rectangular, Circular, dll) → input dimensi dan assign material.
  • Menggambar Frame: Draw > Draw Frame/Cable Element → gambar elemen balok dan kolom mengikuti grid. Gunakan Edit > Replicate untuk menduplikasi lantai.
  • Assign Perletakan (Restraints): Pilih joint pada tumpuan → Assign > Joint > Restraints → pilih tipe: Sendi (Pin: translasi x, y, z terkekang; rotasi bebas), Jepit (Fixed: semua DOF terkekang), Rol (Roller).
  • Definisi Load Pattern: Define > Load Patterns → tambahkan: DEAD (Self-Weight Multiplier = 1), LIVE, WIND, QUAKE, dll.
  • Assign Beban: Pilih elemen/joint → Assign > Frame Loads > Distributed (untuk beban merata/terpusat pada balok) atau Assign > Joint Loads > Forces (untuk beban titik).
  • Definisi Load Combination: Define > Load Combinations → buat kombinasi sesuai SNI (1.2D + 1.6L, 1.2D + 1.0E + L, dll).
  • Run Analysis: Analyze > Set Load Cases to Run → pastikan semua load case aktif → klik Analyze > Run Analysis (F5).
  • Tampilkan Hasil: Display > Show Deformed Shape untuk defleksi; Display > Show Forces/Stresses > Frames untuk diagram gaya dalam (M, V, N).
Hubungan Matriks Kekakuan, Beban & Displacement

Persamaan dasar analisis struktur (Metode Kekakuan / Stiffness Method):

[K] · {D} = {F}

Di mana:

  • [K] = Matriks Kekakuan Global (dirakit dari matriks kekakuan tiap elemen frame, ukuran n×n di mana n = jumlah derajat kebebasan total)
  • {D} = Vektor Displacement Nodal (translasi ux, uy, uz dan rotasi θx, θy, θz di setiap joint)
  • {F} = Vektor Beban Nodal (termasuk konversi beban merata → gaya ekvalen nodal)
Matriks kekakuan elemen: [k]e = (EA/L)[...] + (EI/L³)[...]

SAP2000 secara otomatis merakit [K] global, menerapkan kondisi batas (perletakan), menyelesaikan {D} melalui eliminasi Gauss/dekomposisi LU, lalu menghitung gaya dalam elemen dari displacement yang diperoleh.

Sebutkan langkah-langkah dalam perancangan baja dengan cara auto select penampang sampai mendapatkan hasil penampang yang sesuai dalam SAP2000
Langkah Auto Select Penampang Baja di SAP2000
  • Definisi Material Baja: Define > Materials → pilih atau buat material baja (misal BJ37/ASTM A36: E = 200.000 MPa, fy = 240 MPa).
  • Buat Daftar Auto Select: Define > Section Properties > Frame Sections > Add New Property → pilih Auto Select → beri nama (misal "AUTO_WF") → dari database, masukkan daftar penampang yang akan dikompetisikan (misal: WF 150×75, WF 200×100, WF 250×125, dst.).
  • Assign Auto Select ke Elemen: Pilih semua elemen yang akan didesain → Assign > Frame > Frame Sections → pilih section "AUTO_WF".
  • Set Preferensi Desain Baja: Design > Steel Frame Design > View/Revise Preferences → pilih Design Code (SNI 1729:2020 / AISC 360-16) → set parameter desain (faktor φ, batas rasio L/r, dll).
  • Set Load Combination untuk Desain: Design > Steel Frame Design > Select Design Combos → pilih kombinasi beban ultimit (LRFD) yang akan digunakan.
  • Run Analysis Pertama: Analyze > Run Analysis (F5) — analisis awal dengan penampang default dari list auto select.
  • Mulai Proses Desain: Design > Steel Frame Design > Start Design/Check of Structure → SAP2000 menghitung demand/capacity ratio (D/C) untuk setiap elemen.
  • Review Hasil: Elemen berwarna hijau = OK (D/C ≤ 1.0); merah = overstressed (D/C > 1.0); biru = understressed. Klik elemen untuk melihat detail rasio (aksial, lentur, geser, interaksi).
  • Iterasi Otomatis: Design > Steel Frame Design > Change Design Section atau pilih Reset to Auto Select → SAP2000 memilih penampang terkecil yang memenuhi syarat dari daftar Auto Select.
  • Run Ulang & Verifikasi: Jalankan analisis ulang dengan penampang baru → ulangi proses desain → pastikan semua elemen hijau (D/C ≤ 1.0). Ulangi iterasi jika masih ada elemen merah.
  • Tampilkan Penampang Terpilih: Design > Steel Frame Design > Display Design Info → "Sections Selected" untuk melihat penampang final yang dipilih untuk setiap elemen.
⚠ Catatan: Iterasi biasanya diperlukan 2–3 kali karena perubahan penampang mengubah berat struktur sehingga beban mati sendiri (self-weight) berubah, yang mempengaruhi hasil analisis.

Apa perbedaan dari koordinat lokal dan koordinat global pada element frame, dalam perencanaan apa saja yang mempengaruhi dalam pemodelan di SAP2000, jelaskan dengan gambar?
Koordinat Global vs Koordinat Lokal
Aspek Koordinat Global (X, Y, Z) Koordinat Lokal (1, 2, 3)
Definisi Sistem sumbu tetap untuk seluruh struktur (X: horizontal kanan, Y: horizontal dalam, Z: vertikal atas) Sistem sumbu yang melekat pada setiap elemen frame; berubah mengikuti orientasi elemen
Sumbu X, Y, Z (Cartesian global) Sumbu 1 = arah aksial elemen; Sumbu 2 = tegak lurus di bidang lokal 1-2; Sumbu 3 = tegak lurus di bidang lokal 1-3
Contoh Gravitasi selalu –Z; beban angin searah +X atau +Y Gaya aksial searah sumbu 1; momen lentur di sumbu 3 (major axis)
Menu SAP2000 View > Set Display Options > Show Global Axes Assign > Frame > Local Axes (mengatur sudut angle α)
Gambar 1 — Perbandingan sistem koordinat Global (X,Y,Z) dan Lokal (1,2,3) pada elemen frame miring
Pengaruh dalam Pemodelan SAP2000
  • Pembebanan Elemen (Frame Load): Saat assign beban pada frame, pilih "Local" atau "Global" — beban gravitasi idealnya di koordinat Global (–Z), sedangkan beban tekanan angin tegak lurus elemen di koordinat Lokal (arah 2 atau 3). Menu: Assign > Frame Loads > Distributed → pilih "Coordinate System".
  • Output Gaya Dalam: Hasil gaya dalam selalu dalam koordinat Lokal: P (aksial, sumbu 1), V2, V3 (geser), T (torsi, sumbu 1), M2, M3 (momen). Menu: Display > Show Forces/Stresses > Frames.
  • Orientasi Penampang (Angle): Rotasi penampang terhadap sumbu lokal mempengaruhi arah inersia mayor/minor. Menu: Assign > Frame > Local Axes → ubah sudut Angle (°) untuk memutar penampang, contoh balok dengan sayap menghadap ke bawah.
  • Frame Release: Pelepasan DOF (sendi internal) menggunakan referensi sumbu lokal — misalnya release momen M3 pada ujung batang. Menu: Assign > Frame > Releases/Partial Fixity.
  • Orientasi Element (End Offset): Arah sumbu lokal menentukan arah evaluasi panjang efektif untuk buckling. Menu: Assign > Frame > End (Length) Offsets.

Bagaimana cara memberi beban pelat lantai pada balok?
Cara Pembebanan Pelat Lantai ke Balok

Beban pelat lantai (beban mati finishing + beban hidup) harus didistribusikan ke balok penumpu menggunakan metode tributary area sesuai SNI 1727. Ada dua cara utama:

CARA 1 — Input Manual (Beban Ekivalen pada Balok)
  • Hitung Beban Pelat: Beban total pelat = q (kN/m²) = beban mati pelat (finishing, screed, plafon) + beban hidup. Contoh: q = 1.2 + 2.0 + 1.5 = 4.7 kN/m².
  • Tentukan Distribusi: Berdasarkan rasio sisi ly/lx (panjang/pendek):
    • lx/ly ≤ 0.5: pelat satu arah → semua beban ke balok pendek (merata)
    • lx/ly > 0.5: pelat dua arah → distribusi segitiga (balok pendek) & trapesium (balok panjang)
  • Hitung Beban Ekivalen Segitiga: qek = (q × lx / 3) → beban merata ekivalen untuk balok arah pendek (dengan faktor konversi momen: β = 5/8 × q × lx²).
  • Hitung Beban Ekivalen Trapesium: qek = q × lx/2 × (1 – lx²/(3ly²)) → beban merata ekivalen untuk balok arah panjang.
  • Input ke SAP2000: Pilih balok → Assign > Frame Loads > Distributed → Load Pattern: DEAD atau LIVE → Distribution: Uniform → input nilai beban ekivalen (kN/m) → OK.
CARA 2 — Input Langsung via Area Object (Shell Element)
  • Gambar Pelat: Draw > Draw Rectangular Area / Draw Area → gambar pelat di atas balok perimeter.
  • Assign Beban Area: Pilih area pelat → Assign > Area Loads > Uniform (Shell) → input nilai q (kN/m²).
  • Set Load Transfer: Pilih pelat → Assign > Area > Modify/Show Area Sections → pada tab Stiffness, pilih Membrane saja (tidak diberi kekakuan lentur) agar beban diteruskan ke balok lewat gravitasi; atau gunakan Assign > Area > Modify/Show Mass Source.
  • Gunakan "One Way" atau "Two Way": Untuk pelat sederhana tanpa memodelkan kekakuan, bisa gunakan Assign > Area Loads > Uniform to Frames (Shell) → SAP2000 otomatis mendistribusikan beban area ke balok-balok sekitarnya (One-Way atau Two-Way tergantung setting).
💡 Praktis: Untuk pemodelan rangka sederhana (tanpa shell element pelat), Cara 1 (manual) lebih umum digunakan. Cara 2 digunakan ketika pelat ikut dimodelkan sebagai area object (misalnya pada gedung beton bertulang dengan flat slab).
Gambar 2 — Distribusi beban pelat dua arah: segitiga (balok pendek) dan trapesium (balok panjang)

Sebutkan langkah-langkah secara umum untuk membuat model struktur Truss dengan SAP2000 dan hubungannya dengan matriks kekakuan, beban dan displacement struktur?
Perbedaan Frame vs Truss dalam SAP2000
ℹ Truss (rangka batang) hanya menerima gaya aksial — semua joint berupa sendi (pin). Setiap elemen hanya memiliki 2 DOF translasi per node (tidak ada rotasi). Di SAP2000, ini diimplementasikan dengan Frame Release Momen di kedua ujung setiap elemen.
Langkah-Langkah Pemodelan Truss di SAP2000
  • New Model: File > New Model → pilih satuan → pilih template "2D Truss" atau "Blank" untuk input manual.
  • Definisi Material: Define > Materials → input material baja (E, fy, γ).
  • Definisi Penampang: Define > Section Properties > Frame Sections → pilih profil batang (siku/L, pipa/Pipe, kotak/Box, atau solid rod sesuai desain atap/jembatan).
  • Gambar Elemen Truss: Draw > Draw Frame/Cable Element → gambar semua batang (chord atas, chord bawah, vertikal, diagonal).
  • Assign Semua Ujung sebagai Sendi (Pin): Pilih SEMUA elemen → Assign > Frame > Releases/Partial Fixity → centang Start: Moment 33 (M3) dan End: Moment 33 (M3) → untuk truss 3D, tambahkan M2 juga.
  • Assign Perletakan: Joint tumpuan tetap (Pin): kekang TX, TY, TZ; tumpuan geser (Roller): kekang TY, TZ saja. Gunakan Assign > Joint > Restraints.
  • Input Beban: Beban pada truss biasanya beban titik di joint → Assign > Joint Loads > Forces atau beban merata pada chord atap → Assign > Frame Loads > Distributed.
  • Run Analysis: Analyze > Run Analysis → pastikan Truss tidak ada locking (tidak ada mekanisme).
  • Lihat Hasil: Gaya aksial batang: Display > Show Forces/Stresses > Frames > Axial Force; displacement joint: Display > Show Deformed Shape.
Matriks Kekakuan Elemen Truss

Elemen truss hanya memiliki kekakuan aksial:

[k]e = (EA/L) · [ 1 -1 / -1 1 ] (koordinat lokal)
[K]global = Σ [T]ᵀ [k]e [T] → [K]{D} = {F}

Di mana [T] = matriks transformasi yang bergantung sudut inklinasi elemen (cos α, sin α). Karena tidak ada kekakuan lentur, matriks [K] untuk truss jauh lebih kecil daripada frame. Setelah solver menyelesaikan {D} (displacement nodal), gaya aksial tiap batang dihitung: N = (EA/L) × (D_j − D_i) × cos α.

Sebutkan langkah-langkah dalam perancangan baja sampai mendapatkan hasil penampang yang sesuai dalam SAP2000
Langkah Perancangan Baja di SAP2000

Perancangan baja dapat dilakukan dengan dua cara: (A) Auto Select seperti UTS 2021 Q2, atau (B) Check dengan penampang yang sudah ditentukan. Berikut langkah lengkap dari awal hingga penampang sesuai:

  • Selesaikan Pemodelan & Analysis: Pastikan model sudah lengkap: material, section, beban, load combination, perletakan → Run Analysis berhasil tanpa error.
  • Set Design Code: Design > Steel Frame Design > View/Revise Preferences → pilih Design Code: AISC 360-16 / SNI 1729:2020 → sesuaikan φ (LRFD) atau Ω (ASD).
  • Set Design Combinations: Design > Steel Frame Design > Select Design Combos → centang kombinasi LRFD: 1.2D+1.6L, 1.2D+1.0E+L, dsb.
  • Set Overwrite per Elemen (jika perlu): Pilih elemen → Design > Steel Frame Design > View/Revise Overwrites → ubah faktor panjang efektif K (untuk kolom), batas lendutan, dll.
  • Jalankan Desain: Design > Steel Frame Design > Start Design/Check of Structure.
  • Interpretasi Warna: Hijau (OK, D/C ≤ 1.0) | Merah (Fail, D/C > 1.0) | Biru tua (Understressed, bisa diperkecil).
  • Detail per Elemen: Klik kanan elemen → Steel Design Details → lihat cek aksial, lentur mayor/minor, geser, interaksi, dan controlling equation.
  • Ubah Penampang & Iterasi: Jika merah → pilih penampang lebih besar → Assign > Frame > Frame Sections → Run ulang → Desain ulang. Ulangi hingga semua hijau.
  • Cetak/Export Hasil: Design > Steel Frame Design > Display Design Info → atau File > Print Tables > Steel Frame Design.

Bagaimana cara memberi beban pelat lantai pada balok?

Jawaban untuk soal ini sama dengan UTS 2021 No. 4. Silakan merujuk ke jawaban UTS 2021 Q4 di atas yang mencakup dua cara: (1) Input Manual dengan tributary area dan (2) Input via Area Object. Terdapat tambahan poin penting berikut:

  • Verifikasi Arah Beban: Pastikan beban pelat diinput dalam arah –Z (gravitasi) pada koordinat global. Di menu Distributed Load, set "Direction: Gravity" atau "Direction: –Z".
  • Load Pattern Terpisah: Buat Load Pattern DEAD dan LIVE terpisah — beban mati pelat (finishing, tiles, screed) masuk DEAD; beban hidup lantai (orang, furniture) masuk LIVE. Ini penting untuk kombinasi beban SNI.
  • Beban Dinding di Atas Balok: Beban dinding bata/beton di atas balok dihitung terpisah sebagai UDL (uniform distributed load) di balok: w = γ × t × h (kN/m) dan diinput via Assign > Frame Loads > Distributed.

Sebutkan langkah-langkah secara umum untuk membuat model struktur Frame dengan SAP2000 dan bagaimana mengeluarkan displacement struktur dan internal forces masing-masing batang?
Langkah Pemodelan Frame (ringkasan — lihat UTS 2021 Q1 untuk detail)
  • New Model → Set satuan → Pilih template Frame.
  • Define Materials → Define Frame Sections → Draw Frame Elements.
  • Assign Restraints (perletakan) → Define Load Patterns → Assign Loads.
  • Define Load Combinations → Run Analysis (F5).
Menampilkan Displacement Struktur
  • Deformed Shape Visual: Display > Show Deformed Shape → pilih Load Case/Combo → atur Scale Factor → OK. SAP2000 menampilkan bentuk defleksi dengan warna gradien.
  • Displacement Numerik per Joint: Klik joint yang ingin dilihat → muncul popup nilai U1 (δx), U2 (δy), U3 (δz) dalam mm/cm dan R1, R2, R3 (rotasi dalam rad).
  • Tabel Displacement: Display > Show Tables → pilih Analysis Results → Joint Output → Displacements → pilih Load Case → OK. Tabel menampilkan semua nilai displacement di setiap joint.
  • Export ke Excel: Di jendela tabel, klik File > Export Current Table to Excel atau copy-paste langsung.
  • Cek Lendutan Izin: Bandingkan lendutan maksimum (δmaks) dengan batas izin SNI: δizin = L/240 (beban hidup) atau L/480 (beban total) untuk balok.
Menampilkan Internal Forces (Gaya Dalam)
  • Diagram Gaya Dalam Visual: Display > Show Forces/Stresses > Frames/Cables → pilih komponen: Axial Force (P), Shear 2-2 (V2), Shear 3-3 (V3), Torsion (T), Moment 2-2 (M2), Moment 3-3 (M3) → pilih Load Case → atur skala → OK.
  • Nilai di Titik Tertentu: Klik elemen frame → SAP2000 menampilkan nilai gaya dalam di titik yang diklik, atau klik kanan → Show Forces → detail sepanjang elemen.
  • Tabel Internal Forces: Display > Show Tables → Analysis Results → Element Output → Frame Forces → pilih output stations (jarak pembagian sepanjang elemen) → OK.
  • Notasi Gaya Dalam SAP2000:
    • P = gaya aksial (positif: tarik)
    • V2, V3 = geser lokal arah 2 dan 3
    • T = torsi
    • M2, M3 = momen lentur (M3 = momen mayor untuk profil WF horizontal)
  • Print/Export: File > Print Tables → centang Frame Forces → Print atau Save to File (.txt/.xls).

Sebutkan langkah-langkah dalam optimasi penampang baja sampai mendapatkan hasil penampang yang sesuai dalam SAP2000
Optimasi Penampang Baja — Prosedur Lengkap

Optimasi bertujuan mendapatkan penampang terkecil (paling ekonomis) yang tetap memenuhi semua persyaratan kekuatan dan kekakuan.

  • Buat Daftar Kandidat Penampang: Define > Section Properties > Frame Sections > Add New Property → tipe Auto Select → tambahkan semua kandidat penampang dari database (AISC, SNI) yang relevan, urutkan dari yang terkecil ke terbesar.
  • Assign Auto Select ke Semua Elemen: Pilih elemen → Assign > Frame > Frame Sections → pilih daftar Auto Select. Buat daftar berbeda untuk balok, kolom, dan bracing jika perlu.
  • Tentukan Preferensi & Kombinasi Desain: Buka Design > Steel Frame Design > View/Revise Preferences → set code, φ, dan batasan slenderness (λ_r). Lalu Select Design Combos → pilih kombinasi ultimit.
  • Iterasi 1 — Run Analysis: F5 → analisis dengan penampang awal (atau terbesar dari list jika belum ada basis).
  • Iterasi 1 — Run Steel Design: Design > Steel Frame Design > Start Design/Check of Structure → SAP2000 memilih penampang terkecil yang pass untuk tiap elemen.
  • Iterasi 2 — Update Penampang & Ulangi: Design > Steel Frame Design > Reset All Steel Sections to Auto Select → Run Analysis lagi → Run Steel Design lagi. Penampang baru (lebih kecil/berbeda) mengubah kekakuan dan berat sendiri → hasil bisa berubah.
  • Konvergensi: Ulangi iterasi hingga penampang yang dipilih tidak berubah antar iterasi (biasanya 2–4 iterasi). Ini menandakan solusi konvergen.
  • Cek Lendutan (Serviceability): Setelah kekuatan OK, cek lendutan dengan beban kerja (beban tidak difaktorkan): Display > Show Deformed Shape → pastikan δ ≤ L/240 atau L/360 sesuai fungsi bangunan.
  • Verifikasi Manual: Klik elemen kritis → Steel Design Details → verifikasi rasio D/C, controlling equation, dan kapasitas vs demand.
  • Tampilkan Penampang Terpilih: Design > Steel Frame Design > Display Design Info > Sections Selected → SAP2000 menampilkan nama penampang terpilih di setiap elemen.
Cek Interaksi Aksial-Lentur (AISC 360 / SNI 1729)
Jika Pu/φPn ≥ 0.2: Pu/φPn + (8/9)(Mux/φMnx + Muy/φMny) ≤ 1.0
Jika Pu/φPn < 0.2: Pu/(2φPn) + (Mux/φMnx + Muy/φMny) ≤ 1.0

SAP2000 mengecek persamaan ini secara otomatis dan menampilkan "Controlling Equation" di detail desain. Pastikan nilainya ≤ 1.0 pada semua elemen.

Ada berapa cara input beban pada pelat lantai, sebutkan langkah-langkahnya!
Cara-Cara Input Beban Pelat Lantai di SAP2000

Terdapat 4 cara utama input beban pelat lantai dalam SAP2000:

CARA 1 — Beban Merata Manual pada Balok (UDL/Trapezoidal)
  • Hitung beban pelat total: q = qDL + qLL (kN/m²).
  • Hitung distribusi ke balok menggunakan tributary area (segitiga/trapesium).
  • Input ke balok: Assign > Frame Loads > Distributed → pilih tipe Uniform atau Trapezoidal → input nilai (kN/m).
CARA 2 — Beban Area pada Shell Element (Pelat Dimodelkan)
  • Gambar pelat: Draw > Draw Rectangular Area.
  • Assign section pelat: Assign > Area > Sections.
  • Assign beban: Assign > Area Loads > Uniform (Shell) → input q (kN/m²).
  • Beban otomatis terdistribusi ke elemen-elemen sekitar melalui analisis FEM pelat.
CARA 3 — Beban Area "Uniform to Frames" (Tanpa Kekakuan Pelat)
  • Gambar area/pelat sebagai area object (Null atau Membrane tipis).
  • Pilih area → Assign > Area Loads > Uniform To Frames (Shell).
  • SAP2000 otomatis membagi beban ke frame-frame sekitar berdasarkan tributary one-way atau two-way.
  • Keunggulan: tidak perlu menghitung manual distribusi segitiga/trapesium.
CARA 4 — Beban Titik di Joint (Point Load)
  • Untuk pelat dengan luas tributary kecil yang diwakili satu titik, atau beban kolom di atasnya.
  • Pilih joint → Assign > Joint Loads > Forces → input gaya dalam arah –Z (kN).
  • Cocok untuk beban mesin/peralatan berat yang terpusat.
✅ Rekomendasi: Cara 3 (Uniform to Frames) paling efisien untuk pemodelan gedung karena menghindari perhitungan manual distribusi sambil tetap menjaga model sederhana tanpa kekakuan pelat. Cara 2 (Shell FEM) digunakan jika analisis tegangan pelat juga diperlukan.

Langkah Pemodelan Gedung di SAP2000
  • New Model — Grid & Tinggi Lantai: File > New Model → pilih template "Building" → input jumlah bay (X, Y), lebar bay, jumlah lantai, dan tinggi tiap lantai → OK. Grid otomatis terbentuk.
  • Definisi Material: Define > Materials → buat material baja (E = 200.000 MPa) dan/atau beton (f'c, Ec = 4700√f'c MPa) sesuai kebutuhan.
  • Definisi Penampang Frame: Define > Section Properties > Frame Sections → buat seksi kolom (kolom baja profil H / kolom beton persegi) dan balok (WF / balok beton persegi). Buat variasi ukuran untuk lantai berbeda.
  • Definisi Penampang Pelat: Define > Section Properties > Slab Sections → input tebal pelat dan material → tipe Membrane (hanya distribusi beban) atau Shell Thin/Thick (analisis penuh).
  • Gambar Elemen:
    • Kolom: Draw > Draw Frame → gambar kolom vertikal di semua grid
    • Balok: Draw > Draw Frame → gambar balok horizontal tiap lantai
    • Pelat: Draw > Draw Rectangular Area → gambar di tiap panel lantai
  • Replicate Lantai: Pilih semua elemen lantai 1 → Edit > Replicate → Linear → ΔZ = tinggi lantai → jumlah ulangan = jumlah lantai − 1.
  • Assign Perletakan (Pondasi): Pilih joint di lantai dasar → Assign > Joint > Restraints → pilih jepit (Fixed) untuk pondasi dalam / tiang pancang.
  • Definisi Diaphragm (Opsional): Untuk rigid diaphragm setiap lantai: pilih semua joint satu lantai → Assign > Joint > Diaphragm → buat diaphragm baru (D1, D2, dst).
  • Assign Penampang: Pilih elemen → Assign > Frame > Frame Sections atau Assign > Area > Sections → assign section yang sesuai.
  • Input Semua Beban (lihat tabel beban di bawah).
  • Definisi Load Combinations: Buat kombinasi sesuai SNI 1727 (LRFD): 1.4D, 1.2D+1.6L, 1.2D+1.0E+L, dll.
  • Run Analysis → Review Hasil.

Sebutkan langkah-langkah dalam frame release dan sebutkan alasan dilakukan frame release serta berikan contoh struktur real yang menggunakan frame release?
Apa itu Frame Release?

Frame release adalah pembebasan (pelepasan) derajat kebebasan (DOF) tertentu pada ujung elemen frame, sehingga gaya/momen tertentu tidak dapat ditransfer melalui ujung tersebut. Efeknya adalah membuat kondisi sambungan sendi (pin) pada ujung elemen.

Langkah-Langkah Frame Release di SAP2000
  • Pilih Elemen: Klik atau seleksi elemen frame yang akan diberi release (bisa satu atau beberapa elemen sekaligus).
  • Buka Menu Release: Assign > Frame > Releases/Partial Fixity → muncul dialog Assign Frame Releases.
  • Pilih Ujung & DOF yang Direlease:
    • Start (i): ujung awal elemen
    • End (j): ujung akhir elemen
    DOF yang bisa direlease: Axial (P), Shear 2 (V2), Shear 3 (V3), Torsion (T), Moment 22 (M2), Moment 33 (M3)
  • Centang DOF yang Diinginkan: Untuk sendi momen: centang M3 (atau M2 dan M3 untuk 3D). Untuk sendi penuh: centang semua kecuali P dan V (biasanya tidak diperlukan).
  • Klik OK: SAP2000 akan menandai elemen dengan tanda "•" kecil di ujung yang direlease (terlihat jika display options aktif).
  • Verifikasi: Jalankan analisis → pada diagram momen, elemen dengan release M3 harus menunjukkan momen = 0 tepat di ujung yang direlease.
⚠ Perhatian: Jangan release kedua ujung elemen dengan DOF yang sama sekaligus (misal P di start dan end) — ini akan menyebabkan instabilitas / singular matrix karena elemen menjadi mekanisme (mechanism) yang tidak stabil.
Alasan Menggunakan Frame Release
# Alasan Penjelasan
1 Mewakili Sambungan Sendi Nyata Baut tunggal / sambungan geser saja (shear tab) tidak dapat mentransfer momen. Frame release M3 memodelkan ini dengan akurat.
2 Struktur Truss (Rangka Batang) Semua elemen truss diperlakukan sebagai sendi di kedua ujung sehingga hanya gaya aksial yang bekerja (M3 = 0).
3 Mengurangi Redistribusi Momen Berlebihan Sambungan semi-rigid atau sambungan sederhana tidak dirancang menanggung momen jepit. Release mencegah over-design joint.
4 Stabilitas Numerik Elemen tertentu (misalnya bracing diagonal) lebih akurat dimodelkan dengan release P dan M agar tidak memberikan kekakuan rotasi palsu.

Ada berapa cara input beban pada pelat lantai pada Gedung, sebutkan langkah-langkahnya?
Input Beban Pelat Lantai pada Gedung — 4 Cara

Sama dengan UTS 2023 Q3, terdapat 4 cara. Khusus untuk gedung bertingkat, ada pertimbangan tambahan berikut:

  • Cara 1 — Manual (UDL pada Balok): Hitung tributary area tiap balok → distribusikan q (kN/m²) menjadi w (kN/m) → input via Assign > Frame Loads > Distributed. Cocok untuk model sederhana tanpa pelat sebagai area object.
  • Cara 2 — Shell Element + Uniform Area Load: Gambar pelat sebagai Shell → Assign > Area Loads > Uniform (Shell) → input qDL dan qLL terpisah. SAP2000 mendistribusikan via FEM ke balok-balok. Digunakan jika analisis pelat (defleksi, tulangan pelat) juga diperlukan.
  • Cara 3 — Uniform to Frames: Gambar area Null (area tanpa kekakuan) → Assign > Area Loads > Uniform to Frames → input q → SAP2000 otomatis distribusikan ke balok perimeter dengan tributary. Cara paling cepat dan efisien untuk gedung multi-lantai.
  • Cara 4 — Joint Load (Point Load): Untuk beban terpusat (kolom sub-struktur, mesin berat, tangki air, dll) → Assign > Joint Loads > Forces → input P (kN) arah –Z.
🏢 Khusus Gedung Bertingkat: Perhatikan bahwa beban hidup untuk gedung perkantoran = 2.4 kN/m², hunian = 1.92 kN/m², parkir = 2.4 kN/m², atap datar dapat diakses = 4.8 kN/m² (sesuai SNI 1727:2020). Pisahkan load pattern DEAD, DEAD_ADD (beban mati tambahan), dan LIVE agar kombinasi beban bisa dikontrol.

BAB II — DASAR TEORI
2.1 Prinsip Analisis Rangka Atap (Truss)

Rangka atap dimodelkan sebagai truss (rangka batang), yaitu sistem elemen yang hanya menerima gaya aksial (tarik/tekan) tanpa momen lentur. Hal ini dimungkinkan karena seluruh joint diasumsikan sebagai sendi (pin), dan beban hanya bekerja di titik joint. Persamaan keseimbangan struktur:

[K] · {D} = {F}

Di mana [K] = matriks kekakuan global (dirakit dari kekakuan aksial elemen EA/L), {D} = vektor displacement, {F} = vektor gaya nodal.

2.2 Jenis-Jenis Beban Rangka Atap

a. Beban Mati (DL): Berat sendiri profil baja, berat penutup atap (genteng, spandex, zincalume), berat reng, gording, dan talang. Satuan: kN/m².

b. Beban Hidup Atap (LL): Beban pekerja dan peralatan saat konstruksi/pemeliharaan. Sesuai SNI 1727:2020 minimum 0.96 kN/m² atau 1.0 kN terpusat.

c. Beban Angin (W): Tekanan angin pada bidang atap. Dihitung berdasarkan kecepatan angin dasar (V = 25–35 m/s di Indonesia), koefisien tekanan angin (Cp), faktor exposure, dan faktor topografi. Terdapat tekanan (+) dan hisapan (−) sesuai kemiringan atap.

d. Beban Hujan (R) [jika diminta]: Beban genangan air hujan pada atap datar.
